This module provide a Real-Time insight into what's happening on your website. 

== Description ==
This module in still under development but is working for people who want to try it out. Note that it is possible to write code for your site that will not be compatible with this module, so please test your site and it's features after installing the module to make sure everything is working Ok. If you spot any problems, please deactivate the module and let us know what went wrong.

== Installation ==
Install this module via the control panel, then, visit https://livestats.iflexrts.uk:8001. You will need to be using Google Chrome and be logged in to your Google acount. It will ask you for basic permission to read your identity. Once you have done this, you will need to email support@iflexrts.uk with your google email address and site URL. Wnce approved you will receive an email with a link to the statistics site.
