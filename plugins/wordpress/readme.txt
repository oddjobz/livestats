This module provide a Real-Time insight into what's happening on your website. 

== Description ==
Once installed this module will feed basic information about access to your site to a central logging server. Every time a page is accessed the page URL and details relating to the client (IP address etc) are sent directly from the client browser to the server. You will then be able to connect to the logging server which will give you a live view of what's going on, this includes current connections, sessions, access by country, city, browser etc, all of which are updated in real-time.

Note that it is possible to write code for your site that will not be compatible with this module, so please test your site and it's features after installing the module to make sure everything is working Ok. If you spot any problems, please deactivate the module and let us know what went wrong.

== Installation ==
Install this module via the control panel.

Once installed, please visit;
https://livestats.iflexrts.uk:8001

Requirements;
o you will need to be using Google Chrome
o you will need to be logged logged in to a valid Google acount

You will be asked to allow the site basic permissions to read your identity. Once you have done this, you will need to email support@iflexrts.uk with your google email address and site URL. When approved you will receive an email with a link to the statistics site.
