This module provide a Real-Time insight into what's happening on your website. 

== Description ==
This module in still under development but is working for people who want to try it out. You will first need to install the "iflexRTS SPA" module on order to convert your site into a Single Page Application. Note that it is possible to write code for your site that will not be compatible with the SPA module, so please test your site and it's features after installing the SPA module to make sure everything is working Ok. If you spot any problems, please deactivate the SPA module and let us know what went wrong.

== Installation ==
First, install the iFlexRTS SPA module
Next, install this module
Then, visit https://livestats.iflexrts.uk:8001, you will need to be using Google Chrome and be logged in to your Google acount. It will ask you for basic permission to read your identity. Once you have done this, currently you will need to email support@iflexrts.uk with your google email address and site URL, once approved you will receive an email with a linke to the statistics site.
